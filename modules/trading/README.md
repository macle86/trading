## interval
- 분: m
  - 1m 1분봉, 5m 5분봉, 15m 15분봉
- 시: h
  - 1h 1시간봉, 4h 4시간봉
- 일(하루): d
  - 1d 1일봉
- 주(7일): w
  - 1w: 주봉(1주일)
- 월: M (대소문자 구분 필수, 소문자는 분봉)
  - 1M :1달
